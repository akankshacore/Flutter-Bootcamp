## 💡 Getting Started

To run any project from this repository:

1. Clone the repository
2. Navigate to the specific day's folder (`cd Day 1/`)
3. Run:

   ```bash
   flutter pub get
   flutter run
   ```

Make sure to check the `pubspec.yaml` file for asset declarations before running the app.
